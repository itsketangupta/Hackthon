# Vadathon

Vadathon is a hackthon theme idea based on mumbai vada pav. This is a simple hackathon website template. You can use this template for your hackathon website.

---

## How to run this project locally (or how to clone and modify this project)

### 1. Clone the repository

```bash
git clone https://github.com/itsketangupta/Hackthon.git
```

### 2. Open the project folder

```bash
cd Hackthon
```

### 3. Run the project

This is a simple frontend project, so you can just open the `index.html` file in your browser.

Or, if you are using VS Code:

"Open with Live Server"
